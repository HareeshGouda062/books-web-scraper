# books-web-scraper
Python web scraping project using Requests and BeautifulSoup to extract book title, price, rating, and availability data from books.toscrape.com and save it for analysis.

## Project Overview
This project demonstrates an end-to-end **web scraping workflow** using Python.

The scraper extracts book information from **books.toscrape.com** and prepares the dataset for analysis.

Collected data:
- Title
- Price
- Rating
- Availability

The final dataset is exported to CSV for further data analysis or visualization.
